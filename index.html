FAQ

Preguntas frecuentes

¿Cómo funciona la captura automática desde el correo?
+

Configura las credenciales IMAP de tu correo (Gmail, Outlook u otro) una sola vez. Nexcore IA revisa periódicamente la bandeja de entrada, descarga los XML adjuntos de facturas electrónicas DIAN y los causa automáticamente. También abre ZIPs que contengan XMLs.

¿Funciona con SIESA Enterprise?
+

Sí. Nexcore IA se conecta a la API REST de SIESA con las credenciales de tu empresa. Una vez configurado, las causaciones aprobadas se registran directamente en SIESA sin doble digitación. También es compatible con Siigo Nube y Helisa.

¿El cálculo de retenciones es correcto para 2026?
+

Sí. Nexcore IA usa la UVT 2026 ($52.374), las bases mínimas del Decreto 0572/2025 y las tarifas de ReteICA por municipio actualizadas. Los cálculos se verifican contra las tablas DIAN oficiales y se actualizan cada año fiscal.

¿Puedo subir un Excel con muchas facturas a la vez?
+

Absolutamente. Sube el Excel exportado desde el portal DIAN o el de tu proveedor. Nexcore IA detecta las columnas automáticamente (NIT, base, IVA, fecha), calcula las retenciones para cada fila y las causa en lote con un solo clic.

¿Los datos de mi empresa están seguros?
+

Cada empresa tiene su propia base de datos completamente aislada. Los datos se almacenan con cifrado, backups automáticos cada 6 horas y cumple con la Ley 1581 de 2012 de protección de datos de Colombia.

¿Cuánto tiempo toma la configuración inicial?
+

Menos de 20 minutos. Solo necesitas configurar las cuentas PUC de tu empresa, las credenciales SIESA y el correo IMAP. A partir de ahí el sistema trabaja solo. No requiere instalación ni conocimientos técnicos.
